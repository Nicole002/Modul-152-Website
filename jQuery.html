Was ist jQuery?

Mit jQuery ist es viel einfacher JavaScrip in die Webseite einzubinden, da man mit jQuery ganze JavaScript methoden mit einer Zeile aufrufen kann.

Auf der Webseite einbinden

Um jQuery auf der Webseite einzubinden, muss das jQuery selbst bevor dem jQuery-Code eingebunden werden, da sonst eine Fehlermeldung auftritt.

jQuery Syntax

Der Syntax ist dafür zugeschnitten, HTML Elemente auszusuchen und dann eine Aktion auszuführen.

Basic Syntax: $(selector).action()

Beispiele:

$(this).hide() - hides the current element.

$("p").hide() - hides all p elements.
