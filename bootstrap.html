Bootstrap

Download und Installation

Um Bootstrap zu downloaden geht man auf die Webseite von Bootstrap. Dort hat es einen Knopf auf welchem "Download" steht. Wie man es installiert habe ich keine ahnung lol.

Grid-System

Mit Bootstrap kann man bis zu 12 "columns" benutzen. Dies erleichtert einem das Design, da alles strukturiert ist und es mit felxbox arbeitet, was heisst, dass es responsive ist.

Einzelne Komponenten einfügen

Um einzelne Komponenten einzufügen sucht man sich zuerst etwas auf der Bootstrap Webseite unter "Examples" aus und klickt dies an. Rechtsklick irgendwo auf der Webseite und "Seitenquelltext anzeigen" klicken. Dies liefert einem das HTML. Nun muss man diese Komponenten suchen, kopieren und einfach im eigenen HTML file einfügen. Man muss jedoch beachten, ob man zusätzlich ein CSS oder JQuery file einfügen muss.
