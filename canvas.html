HTML5 Canvas

Was ist canvas?

canvas ist ein HTML5 Element, auf das man mit Hilfe von Skripten Animationen, Grafiken und Bilder projizieren kann.

Grundlagen

Das canvas-Element hat die Breite von 300px und eine Höhe von 150px, wenn nichts anderes gesetzt wurde. Man soll dem canvas-Elemtn eine id geben, da dies den zugriff vereinfacht. Mit jQuery kann man canvas bearbeiten:

var canvas = document.getElementById("tutorial") = speichert in der Variablen canvas den DOM-Knoten

Rechteck zeichnen

Es gibt drei Funktionen, welche auf verschiedenste Art Rechtecke zeichnen:
fillRect(x, y, breite, höhe)
Zeichnet ein gefülltes Rechteck

strokeRect(x, y, breite, höhe)
Zeichnet den Rahmen eines Rechteckes

clearRect(x, y, breite, höhe)
Der Bereich des Rechteckes wird transparent
